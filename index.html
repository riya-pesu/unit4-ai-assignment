Overview

This repository provides a compact, well-documented bubble sort implementation in Python (bubble_sort.py). It's intended for learning and small experiments rather than production use.

Real World Applications of Bubble Sort

Education & Demonstration

Bubble sort is simple and easy to understand, making it ideal for teaching basic sorting concepts, algorithm tracing, and demonstrating algorithmic complexity.

Small, Nearly-Sorted Lists

For very small or nearly-sorted lists, bubble sort can be acceptable and can exit early when no swaps occur.

Embedded / Constrained Environments

In very constrained systems where code simplicity and predictable behavior are more important than raw speed, bubble sort's small code size can be helpful.

Code Explanation

The following is the bubble_sort.py implementation used in this project. It includes a small CLI, a parser for numeric tokens, and clear error messages when elements cannot be compared.

```python
#!/usr/bin/env python3
"""
bubble_sort.py

Concise, readable bubble sort with minimal surface area:
- bubble_sort(sequence, reverse=False, in_place=False)
- bubble_sort_in_place(list, reverse=False)
- small CLI for quick use

Preserves helpful TypeError messages for non-iterable inputs and
non-comparable elements while keeping the implementation compact.
"""
from typing import Iterable, List, Any
import argparse
import sys

__all__ = ["bubble_sort", "bubble_sort_in_place"]


def _parse_token(tok: str) -> Any:
    t = tok.strip()
    for conv in (int, float):
        try:
            return conv(t)
        except Exception:
            continue
    return t


def bubble_sort_in_place(arr: List[Any], *, reverse: bool = False) -> List[Any]:
    """In-place bubble sort. Returns the same list after sorting."""
    if not isinstance(arr, list):
        raise TypeError("bubble_sort_in_place requires a list for in-place sorting.")
    n = len(arr)
    for end in range(n - 1, 0, -1):
        swapped = False
        for i in range(end):
            try:
                if (arr[i] > arr[i + 1]) ^ reverse:
                    arr[i], arr[i + 1] = arr[i + 1], arr[i]
                    swapped = True
            except TypeError as exc:
                raise TypeError(f"Cannot compare {arr[i]!r} and {arr[i+1]!r}") from exc
        if not swapped:
            break
    return arr


def bubble_sort(sequence: Iterable[Any], *, reverse: bool = False, in_place: bool = False) -> List[Any]:
    """
    Sort sequence using bubble sort.
    - If in_place=True, sequence must be a list and will be mutated and returned.
    - Otherwise a new sorted list is returned.
    """
    if not hasattr(sequence, "__iter__"):
        raise TypeError("sequence must be iterable")
    if in_place:
        return bubble_sort_in_place(sequence, reverse=reverse)  # type: ignore[arg-type]
    return bubble_sort_in_place(list(sequence), reverse=reverse)


def _cli(argv: List[str]) -> int:
    p = argparse.ArgumentParser(description="Bubble sort (concise).")
    p.add_argument("items", nargs="*", help="Items to sort (space separated).")
    p.add_argument("-r", "--reverse", action="store_true", help="Sort descending.")
    p.add_argument("-v", "--verbose", action="store_true", help="Show original + sorted.")
    args = p.parse_args(argv)

    if not args.items:
        try:
            raw = input("Enter items (comma separated): ").strip()
        except (EOFError, KeyboardInterrupt):
            return 1
        if not raw:
            return 1
        tokens = [t for t in raw.split(",")]
    else:
        tokens = args.items

    items = [_parse_token(t) for t in tokens]
    try:
        out = bubble_sort(items, reverse=args.reverse, in_place=False)
    except TypeError as exc:
        print(f"Error: {exc}", file=sys.stderr)
        return 2

    if args.verbose:
        print("Original:", items)
        print("Sorted:  ", out)
    else:
        print(" ".join(map(str, out)))
    return 0


if __name__ == "__main__":
    raise SystemExit(_cli(sys.argv[1:]))
```

Usage

Clone the repository and run the script from your shell:

```bash
# clone the repo
git clone https://github.com/riya-pesu/unit4-ai-assignment.git
cd unit4-ai-assignment

# run with positional items
python3 bubble_sort.py 5 1 4 3 2

# run descending
python3 bubble_sort.py --reverse 5 1 4 3 2

# verbose output
python3 bubble_sort.py -v 3 2 1

# interactive (comma-separated prompt)
python3 bubble_sort.py
# then enter: 3, 1, 2
```

Tip: The script attempts to parse tokens as integers or floats automatically. If comparison fails (e.g., mixing incomparable types), a helpful error is printed.
